# coc_discord_bot

## Requirements
It is required to have a .env file in the src directory containing three environment variables:
* DISCORD_TOKEN
* COC_TOKEN
* COC_CLANTAG

    for example:
    `DISCORD_TOKEN = '<discord_bot_token>'`
    `COC_TOKEN = '<coc_api_token>'`
    `COC_CLANTAG = '<clantag>'`
